Instructions

CSV File Format

Your CSV file should contain the following columns:

  • linkedin_url (required): Full LinkedIn profile URL
  • name (optional): Person's name
  • company (optional): Current company
  • notes (optional): Additional notes
Important Notes
  • All URLs must be valid LinkedIn profile URLs
  • Duplicate profiles will be ignored
  • Processing may take a few minutes for large lists
  • You will be notified by email when processing is complete
